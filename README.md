## 🌟 Sobre o Projeto

Este é o meu portfólio pessoal construído para demonstrar minhas habilidades, projetos e formas de contato. Ele apresenta um design moderno, responsivo e com animações fluidas para proporcionar a melhor experiência de usuário. Além de contar com áreas detalhadas sobre a minha trajetória profissional e as tecnologias que domino.

### ✨ Principais Funcionalidades

- **Navegação Dinâmica**: Sistema de rotas inteligente utilizando o `react-router-dom`.
- **Animações e Efeitos**: Transições de página e componentes usando `framer-motion` e um efeito incrível de "Matrix Background" (`MatrixBackground.jsx`).
- **Páginas Completas e Integradas**:
  - `🏠 FirstPage`: Página inicial de impacto.
  - `👤 AboutPage`: Sobre mim e minha história.
  - `💼 MyProjects`: Galeria interativa com os meus melhores projetos.
  - `💻 TechPage`: Minhas habilidades e tecnologias utilizadas no dia a dia.
  - `📄 ResumePage`: Meu currículo profissional detalhado.
  - `📱 ContactPage` & `💰 OrcamentoPage`: Para contato direto e propostas de orçamento.
- **Estilização Elegante**: Uso intenso de **Sass (SCSS)** e componentes do **Chakra UI** para criar um layout visualmente atrativo e com suporte a temas dinâmicos (modo claro/escuro) via `next-themes`.
- **Carrosséis Interativos**: Exibição elegante de interfaces e galerias através da biblioteca `swiper`.

## 🚀 Como Executar o Projeto

Siga os passos abaixo para rodar o projeto na sua máquina local.

### Pré-requisitos
Você precisará ter o [Node.js](https://nodejs.org/en/) e um gerenciador de pacotes (`npm` ou `yarn` / `pnpm`) instalados em sua máquina.

### Passo a Passo da Instalação

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/SeuUsuario/react-portfolio.git
   ```

2. **Acesse o diretório do projeto:**
   ```bash
   cd react-portfolio
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```
   *(Ou use `yarn install` / `pnpm install` dependendo do seu gerenciador)*

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

5. O projeto estará rodando magicamente na sua máquina! Acesse [http://localhost:5173/](http://localhost:5173/) (ou na porta indicada pelo Vite) no seu navegador.
